> Acoustic monitoring on Raspberry Pi. Camera species ID. Hive health. Pesticide mapping.

## The Solution

**AcousticMonitor** — Raspberry Pi + microphone pollinator detection (Buzzdetect-style)\n**SpeciesIdentifier** — Camera-based species ID from phone photos\n**HiveHealthMonitor** — Weight, temperature, acoustic analysis of beehive status\n**PesticideMapper** — Correlate spray schedules with pollinator activity zones\n**CitizenScience** — Aggregate and quality-control community observation data
